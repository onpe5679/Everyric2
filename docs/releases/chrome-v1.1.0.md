# Everyric Chrome 확장 1.1.0

유튜브 영상 위에 타임싱크 가사·번역·한글 독음·가라오케 음정 바를 얹어 주는 Chrome 확장의 첫 GitHub 릴리즈입니다.

## 주요 기능

- **가사 오버레이** — 유튜브 위 드래그 가능한 패널, 현재 줄 하이라이트 + 글자 단위 카라오케 필
- **자동 가사 검색** — 서버 저장 싱크 → 보카로 가사 위키 → LRCLIB 순
- **AI 싱크 생성** — 가사 붙여넣기 → 서버가 오디오를 받아 CTC 정렬 (진행 칩 표시, 클릭으로 취소)
- **번역·한글 독음** — LLM이 곡 전체 맥락으로 가사체 번역 + 원문 발음의 한글 표기
- **가라오케 음정 바 (BETA)** — PiP 창에 멜로디 노트·계이름·마이크 음정 궤적, 멜로디 신디사이즈·메트로놈 재생
- **싱크 링크·영상별 오프셋** — inst/커버 영상이 원본 전사를 재사용, ±0.1s 오프셋 영상별 저장

## 설치

> Chrome은 웹스토어 외부 확장의 일반 설치를 막기 때문에 개발자 모드로 로드합니다.

1. 아래 **Assets**에서 `Everyric-Chrome-1.1.0.zip`을 받아 압축을 풉니다.
2. `chrome://extensions` 접속 → 우상단 **개발자 모드** 켜기.
3. **압축해제된 확장 프로그램 로드** → 압축 푼 폴더 선택.
4. 유튜브에서 음악 영상을 열면 패널이 뜹니다 (툴바 아이콘으로 토글).

## 요구 사항

| 항목 | 내용 |
|---|---|
| 브라우저 | Chrome 103+ (Manifest V3) |
| 서버 | [Everyric2 서버](https://github.com/onpe5679/Everyric2#빠른-시작) 실행 필요 — 가사 검색·싱크 생성·번역의 백엔드 |
| 서버 URL | 기본 `http://127.0.0.1:8000` (Windows에서 `localhost` 대신 127.0.0.1 권장) |

## 알려진 제한

- 개발자 모드 로드 방식이라 자동 업데이트가 없습니다 — 새 버전은 zip을 다시 받아 교체하세요.
- 서버 없이 열면 LRCLIB 검색 등 일부 기능만 동작합니다.
